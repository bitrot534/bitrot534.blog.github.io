AGE-1.2.0 on OBSD 7.6

AGE is an Opensource, modern, minimal encryption option.

Encryption is way to protect data. 2 methods include: Symmetric Encryption and Asymmetric Encryption. Symmetric relies on a password for protection, while Asymmetric uses a pair of keys, public key and private key. Whatever method you pick, security comes down to a very strong password as weak passwords are compromised. Test out until confortable and think of secure ways to store password locally as well as remotely.

[ SYMMETRIC - ENCRYPT FILE W/ PASSWD ]

$age -p -o password.age password.txt

[ SYMMETRIC - DECRYPT FILE W/ PASSWD TO SDOUT ]

$age -d password.age

[ SYMMETRIC - DECRYPT TO FILE ]

$age -d key.age password.txt.age > password.txt

-----

[ ASYMMETRIC - MAKE PASSWORD PROTECTED AGE KEY AND CREAT THE 2 KEYS ]

$age-keygen | age -p .key age

Public key:agelyhm4gctwfmrpz87tds1m550wrx6m79y9f2hdzt01ndjnehwjOukqrjpyx5

[ ASYMMETRIC - ENCRYPT FILE ]

$age -r agelyhm4gctwfmrpz87tds1m550wrx6m79y9f2hdzt01ndjnehwjOukqrjpyx5 password.txt > password.txt.age (the long string of random data in this example is the public key from above)

[ ASYMMETRIC - DECRYPT AGE KEY FILE ]

$age -d key.age

[ ASYMMETRIC - DECRYPT TO STOUT ]

$age -d -i key.age password.txt.age

[ ASYMMETRIC - DECRYPT TO FILE ]

$age -d -i key.age password.txt.age > password.txt>
