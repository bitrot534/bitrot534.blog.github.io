AGE is an Opensource encryption option that can replace the older GnuPG.

Encryption is way to protect data. 2 methods include: Symmetric Encryption and Asymmetric Encryption. Symmetric relies on a password for protection, while Asymmetric uses a pair of keys: a public key and a private key. Whatever method you pick, security comes down to a very strong password as weak passwords are compromised.

[ Relevant Documentation ]

Options:

-d, --decrypt Decrypt the input to the output.

-o, --output OUTPUT Write the result to the file at path OUTPUT.

-p, --passphrase Encrypt with a passphrase.

-r, --recipient RECIPIENT Encrypt to the specified RECIPIENT. Can be repeated.

-i, --identity PATH Use the identity file at PATH. Can be repeated.

[ Symmetric ]

Encrypt file w/ passwd

$age -p -o document.age document.txt

Decrypt file to SDOUT

$age -d document.age

Decrypt to file

$age -d -o document.txt document.age

[ Asymmetric ]

Create AGE pub and priv keys w/ passwd

$age-keygen | age -p -o key.age

Enter passphrase (leave empty to autogenerate a secure one):

Encrypt file to yourself

$age -r (insert-your-public-key-here) -o document.age document.txt

Decrypt AGE private key to SDOUT

$age -d key.age

Decrypt a file to SDOUT

$age -d -i key.age document.txt.age

Decrypt to file

$age -d -i key.age -o document.txt document.txt.age
