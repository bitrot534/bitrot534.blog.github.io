AGE-1.2.0 on OBSD 7.6

AGE is an Opensource encryption option that can replace the older GnuPG.

Encryption is way to protect data. 2 methods include: Symmetric Encryption and Asymmetric Encryption. Symmetric relies on a password for protection, while Asymmetric uses a pair of keys: a public key and a private key. Whatever method you pick, security comes down to a very strong password as weak passwords are compromised.

[ SYMMETRIC ]

  • ENCRYPT FILE W/ PASSWD

    • $age -p -o document.age document.txt

  • DECRYPT FILE W/ PASSWD TO SDOUT

    • $age -d document.age

  • DECRYPT TO FILE

    • $age -d document.txt.age > document.txt

    [ ASSYMMETRIC ]

  • ENCRYPT WITH PUB AND PRIV KEYS

    • $age-keygen | age -p .key age

    Public key:agelyhm4gctwfmrpz87tds1m550wrx6m79y9f2hdzt01ndjnehwjOukqrjpyx5

  • ENCRYPT FILE

    • $age -r *Insert Public Key* document.txt > document.txt.age

  • DECRYPT AGE KEY FILE

    • $age -d key.age

  • DECRYPT TO STOUT

    • $age -d -i key.age document.txt.age

  • DECRYPT TO FILE

    • $age -d -i key.age document.txt.age > document.txt
